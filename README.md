# Тестовое задание
## Запуск программы
Склонировать репозиторий

```
git clone https://github.com/MarienkoKate/testYadro.git
```
Перейти в директорию

```
cd testYadro
```
Скомпилировать файлы

```
g++ -o main Main.cpp Cafe.cpp
```

Запуск

```
./main file.txt
```
